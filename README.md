# Vinayaka AI WhatsApp Admission Agent

AI-powered WhatsApp admission assistant built using OpenAI API, n8n, MongoDB, and WhatsApp Cloud API.

## Features

- Automated student enquiry handling
- Lead collection workflow
- Conversational AI responses
- Follow-up automation
- MongoDB-backed memory
- Production-ready workflow automation

## Tech Stack

- OpenAI API
- n8n
- WhatsApp Cloud API
- MongoDB
- Node.js
- Express.js

## Architecture

1. Student sends WhatsApp message
2. WhatsApp Cloud API receives webhook
3. n8n processes workflow
4. OpenAI generates intelligent response
5. MongoDB stores conversational memory
6. Automated follow-up triggered

⚠️ Source code is currently private while deployment and optimization are in progress.
